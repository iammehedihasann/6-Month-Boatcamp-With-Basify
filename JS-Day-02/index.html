Arithmatic And Operator

  1. + : Addition : 5+3=8

  2. - : Subtraction : 5-3=2

  3. * : Multiplication : 5*3=15

  4. / : division : 6/3=2

  5. % : modulas : 5+3=2

  6. ++ : increment:

  7. --: decrement :

Assignment Operators

  1. = : Assigns a value : a=8

  2. += : Adds and assigns : a+=2

  3. -= : Subtracts and assigns : a-=15

  4. *= : Multiplies and assigns : a*=2

  5. /= : Divides and assigns : a/=2

  6. %= : Modulus and assigns: a %=2

Comparison Operators

  1. == : Equal t0: 5 = '5'

  2. === : Strict equal (same type) 5 === '5'

  3. != : Not equal 5 != '5'

  4. !== : Strict not equal 5 !== '5'

  5. > : Greater than 5 > 3

  6. < : Less than 5 < 3

  7. >= : Greater than or equal to 5 >= 5

  8. <= : Less than or equal to 5 <= 3

Logical Operators

  1. && : Logical and: true && false

  2. ` : Logical OR :`

  3. ! : Logical NOT: !true
