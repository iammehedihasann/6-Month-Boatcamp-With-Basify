Fixed Position

This element is fixed to the viewport. It remains in the same position even when the page is scrolled.

Static Position

This is a statically positioned element. By default, all elements are positioned static unless otherwise specified. It flows naturally with the document.

Relative Position

This is a relatively positioned element. It remains in the document flow but can be offset from its normal position using top, right, bottom, or left properties.

Absolute Position

This element is positioned absolutely relative to the nearest positioned ancestor. It is taken out of the document flow, and other elements don't affect its placement.
